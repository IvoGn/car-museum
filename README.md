# Car Museum Website

## Description

This project is a simple static website representing a virtual car museum.  
It was developed as an early project during my web development training.

The website showcases different cars and provides basic information about exhibits in a clean and structured layout.

---

## Features

- Display of car exhibits  
- Structured layout with multiple pages  
- Basic navigation between sections  
- Use of HTML, CSS, and JavaScript  
- Integration of Bootstrap for styling  

---

## Technologies Used

- HTML5  
- CSS3 / SCSS  
- JavaScript  
- Bootstrap  

---

## Getting Started

To run the project locally:

1. Clone the repository:

   git clone https://github.com/IvoGn/car-museum.git

2. Open the project folder:

   cd car-museum

3. Open index.html in your browser.

---

## Purpose

This project was created to practice fundamental web development skills, including layout design, responsive styling, and basic JavaScript functionality.
